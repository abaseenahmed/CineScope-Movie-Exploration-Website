# 🎬 CineScope – Smart Movie Explorer

A modern movie discovery web app built using React & Tailwind CSS, powered by OMDb API.

## 🚀 Features
- 🔍 Smart search with debounce
- 🎥 Movie details page
- ❤️ Favorites system (localStorage)
- 🌙 Dark/Light mode
- 📊 Filters & sorting
- 🔄 Pagination

## 🛠 Tech Stack
- React
- Tailwind CSS
- Axios
- React Router
